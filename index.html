This is a website I built for myself, to practice ひらがな (hiragana), a Japenese Symbol system.

The basic rules

You'll get a romanized symbol, and the objective is to draw it from memory. Once you're done, click "answer" and it'll show you what it should have been.

Hopefully it'll help you get better at not only recognizing the symbols, but also remembering on how to draw them.

Setup

Start (No Dakuon) Start (Dakuon Included) Start (Everything)

Dakuon

Dakoun is a symbol group of ひりがな, that includes accented ひりがな symbols. Examples include: が,ぎ,ぐ,ぜ,ぞ

Everything

This includes everything. Dakoun, and Handakuten (combo wit others). Examples include: きゃ,きゅ, and みょ
